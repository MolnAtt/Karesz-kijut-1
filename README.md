# Karesz kijut I.
Karesznek ki kell jutnia a pálya szélére. Álljon ki a pálya szélére és szívjon egy kis friss levegőt! A pálya széle véletlenszerűen van elhelyezve a négy égtáj fele, ahogy Karesz kezdeti pozíciója is véletlenszerű. Ebben a feladatban nem lehet ultrahang szenzorokat használni!
